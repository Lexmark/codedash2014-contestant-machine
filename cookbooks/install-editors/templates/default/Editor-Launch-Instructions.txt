#Opening Eclipse for Java
1. Open a terminal (press Ctrl + Alt + T)
2. Type eclipse-java
3. Hit enter

OR

1. Click the unity launcher icon
2. Search for eclipse-java
3. Click on the "Eclipse Java" icon.
